1. Selected Models — Why These Are the Best Choice
Every model in the pipeline was chosen for a specific reason. Below is the full justification for each selection, their advantages, and why they outperform alternatives for a school attendance system.

1.1  SCRFD 500M — Face Detection

Why SCRFD 500M is the right choice:
Runs at 25 FPS on CPU — no GPU needed, perfect for school hardware budget
Detects all students in one frame simultaneously — essential for a classroom with 30+ students
Returns 5 facial landmarks (eye corners, nose tip, mouth corners) which enables precise face alignment before recognition, boosting ArcFace accuracy significantly
Much faster and lighter than alternatives: RetinaFace (slower), MTCNN (slower, less accurate on small faces), Haar Cascade (outdated, poor accuracy)
Part of InsightFace ecosystem — perfect compatibility with ArcFace R50 used in Stage 3

1.2  MiniFASNet V2 — Liveness Detection

Why MiniFASNet V2 is the right choice:
Passive detection — students just look at the camera. No blink test, no head movement required. Critical for classroom convenience where 30 students need quick recognition
Analyzes Fourier frequency spectrum of face texture — catches print attacks and screen attacks that fooled older models
Ultra-lightweight (1.8 MB) — barely impacts performance when running alongside SCRFD and ArcFace
Works with two-model ensemble: combine with MiniFASNetV2SE for even higher accuracy (average both scores)
Outperforms DeepPixBiS and FaceDs in speed while maintaining comparable accuracy on anti-spoofing benchmarks

1.3  ArcFace R50 — Face Recognition

Why ArcFace R50 is the right choice:
Industry gold standard — used in production systems worldwide, extensively tested
ArcFace loss function maximizes the angular margin between different identities, producing embeddings that are far apart for different people and very close for the same person — ideal for high-accuracy matching
99.77% accuracy means almost no false identity matches in practice
ONNX format available — works directly with ONNX Runtime, no PyTorch or TensorFlow required
512-dim embeddings are compact enough to store for hundreds of students and fast to compare using cosine similarity
Outperforms FaceNet (99.63%), CosFace (99.73%), and SphereFace (99.42%) on standard benchmarks

Step 1.1 — Face Detector (pipeline/detector.py)
import insightface
import numpy as np
 
class FaceDetector:
    def __init__(self):
        self.app = insightface.app.FaceAnalysis(
            name='buffalo_sc',
            allowed_modules=['detection']
        )
        self.app.prepare(ctx_id=0, det_size=(640, 640))
 
    def detect(self, frame):
        """Returns list of Face objects with .bbox and .kps"""
        return self.app.get(frame)
 
    def crop_face(self, frame, face, padding=0.25):
        """Crop face with padding for liveness + recognition"""
        x1,y1,x2,y2 = face.bbox.astype(int)
        pad_x = int((x2-x1)*padding)
        pad_y = int((y2-y1)*padding)
        x1 = max(0, x1-pad_x); y1 = max(0, y1-pad_y)
        x2 = min(frame.shape[1], x2+pad_x)
        y2 = min(frame.shape[0], y2+pad_y)
        return frame[y1:y2, x1:x2]

Step 1.2 — Liveness Detector (pipeline/liveness.py)
import onnxruntime as ort
import numpy as np, cv2
 
class LivenessDetector:
    def __init__(self, model_path: str, threshold=0.70):
        self.sess = ort.InferenceSession(model_path,
                      providers=['CPUExecutionProvider'])
        self.threshold = threshold
 
    def check(self, face_crop) -> tuple[bool, float]:
        img = cv2.resize(face_crop, (80, 80))
        img = img.astype(np.float32) / 255.0
        img = (img - [0.485,0.456,0.406]) / [0.229,0.224,0.225]
        img = np.transpose(img,(2,0,1))[np.newaxis].astype(np.float32)
        out = self.sess.run(None, {self.sess.get_inputs()[0].name: img})[0][0]
        e = np.exp(out - out.max())
        live_score = float(e[1] / e.sum())
        return live_score >= self.threshold, live_score

Step 1.3 — Face Recognizer with FAISS (pipeline/recognizer.py)
import insightface, faiss, numpy as np, pickle, os
 
class FaceRecognizer:
    def __init__(self, threshold=0.55, index_path='embeddings/faiss.index',
                 names_path='embeddings/names.pkl'):
        self.app = insightface.app.FaceAnalysis(
            name='buffalo_l', allowed_modules=['recognition'])
        self.app.prepare(ctx_id=0)
        self.threshold = threshold
        self.index_path = index_path
        self.names_path = names_path
        self.index = None
        self.names = []
        self._load()
 
    def _load(self):
        if os.path.exists(self.index_path):
            self.index = faiss.read_index(self.index_path)
            with open(self.names_path,'rb') as f:
                self.names = pickle.load(f)
        else:
            self.index = faiss.IndexFlatIP(512)  # Inner product = cosine on L2-norm
 
    def add_student(self, name: str, embeddings: list):
        avg = np.mean(embeddings, axis=0).astype(np.float32)
        avg /= np.linalg.norm(avg)   # L2 normalize
        self.index.add(avg[np.newaxis])
        self.names.append(name)
        faiss.write_index(self.index, self.index_path)
        with open(self.names_path,'wb') as f: pickle.dump(self.names, f)
 
    def get_embedding(self, face_img):
        faces = self.app.get(face_img)
        if not faces: return None
        return faces[0].normed_embedding
 
    def identify(self, face_img) -> tuple[str, float]:
        if self.index.ntotal == 0: return 'Unknown', 0.0
        emb = self.get_embedding(face_img)
        if emb is None: return 'Unknown', 0.0
        emb = emb[np.newaxis].astype(np.float32)
        scores, indices = self.index.search(emb, 1)
        score = float(scores[0][0])
        if score >= self.threshold:
            return self.names[indices[0][0]], score
        return 'Unknown', score

Step 1.4 — Attendance Logic (pipeline/attendance_logic.py)
from datetime import date
from database import get_db_connection
 
def mark_attendance(student_id: int, confidence: float, camera_id: str) -> bool:
    """
    Returns True if attendance was marked, False if already marked today.
    UNIQUE KEY on (student_id, date) prevents any duplicate inserts.
    """
    conn = get_db_connection()
    cursor = conn.cursor()
    try:
        cursor.execute(
            '''INSERT IGNORE INTO attendance
               (student_id, date, confidence, camera_id)
               VALUES (%s, %s, %s, %s)''',
            (student_id, date.today(), confidence, camera_id)
        )
        conn.commit()
        return cursor.rowcount > 0   # 0 = duplicate, 1 = new entry
    finally:
        cursor.close(); conn.close()
 
def get_student_id_by_name(name: str) -> int | None:
    conn = get_db_connection()
    cursor = conn.cursor()
    cursor.execute('SELECT id FROM students WHERE name=%s AND is_active=1', (name,))
    row = cursor.fetchone()
    cursor.close(); conn.close()
    return row[0] if row else None

Step 2.2 — Process Frame Endpoint
@app.post('/api/process-frame')
async def process_frame(file: UploadFile = File(...),
                        camera_id: str = Form('cam_01')):
    data = await file.read()
    frame = cv2.imdecode(np.frombuffer(data, np.uint8), cv2.IMREAD_COLOR)
    results = []
 
    faces = detector.detect(frame)
    for face in faces:
        crop = detector.crop_face(frame, face)
 
        # Stage 2: Liveness
        is_live, live_score = liveness.check(crop)
        if not is_live:
            results.append({'status': 'spoof', 'live_score': live_score})
            continue
 
        # Stage 3: Recognition
        name, confidence = recognizer.identify(crop)
        if name == 'Unknown':
            results.append({'status': 'unknown', 'confidence': confidence})
            continue
 
        # Mark attendance
        student_id = get_student_id_by_name(name)
        marked = mark_attendance(student_id, confidence, camera_id)
        results.append({
            'name': name,
            'status': 'marked' if marked else 'already_marked',
            'confidence': round(confidence, 3),
            'live_score': round(live_score, 3)
        })
 
    return {'faces_detected': len(faces), 'results': results}

Step 2.3 — Student Registration Endpoint
@app.post('/api/register')
async def register_student(
    name:        str  = Form(...),
    roll_number: str  = Form(...),
    class_name:  str  = Form(...),
    photos:      list[UploadFile] = File(...)
):
    # Check duplicate by roll number
    conn = get_db_connection()
    cursor = conn.cursor()
    cursor.execute('SELECT id FROM students WHERE roll_number=%s', (roll_number,))
    if cursor.fetchone():
        return {'success': False, 'error': 'Student already registered'}
 
    embeddings = []
    for photo in photos:
        data = await photo.read()
        img = cv2.imdecode(np.frombuffer(data, np.uint8), cv2.IMREAD_COLOR)
        faces = detector.detect(img)
        if faces:
            crop = detector.crop_face(img, faces[0])
            is_live, _ = liveness.check(crop)
            if is_live:
                emb = recognizer.get_embedding(crop)
                if emb is not None: embeddings.append(emb)
 
    if len(embeddings) < 5:
        return {'success': False, 'error': f'Only {len(embeddings)} valid live faces found. Need at least 5.'}
 
    # Save to MySQL
    cursor.execute(
        'INSERT INTO students (name,roll_number,class_name) VALUES (%s,%s,%s)',
        (name, roll_number, class_name))
    student_id = cursor.lastrowid
    conn.commit(); cursor.close(); conn.close()
 
    # Save embeddings to FAISS
    recognizer.add_student(name, embeddings)
    return {'success': True, 'student_id': student_id, 'samples_used': len(embeddings)}

@app.get('/api/attendance')
def get_attendance(date_str: str = None, class_name: str = None):
    conn = get_db_connection()
    cursor = conn.cursor(dictionary=True)
    query = '''SELECT s.name, s.roll_number, s.class_name,
                      a.date, a.marked_at, a.confidence, a.camera_id
               FROM attendance a JOIN students s ON a.student_id=s.id
               WHERE 1=1'''
    params = []
    if date_str: query += ' AND a.date=%s'; params.append(date_str)
    if class_name: query += ' AND s.class_name=%s'; params.append(class_name)
    query += ' ORDER BY a.marked_at DESC'
    cursor.execute(query, params)
    rows = cursor.fetchall()
    cursor.close(); conn.close()
    return rows
 
@app.get('/api/attendance/export')
def export_attendance(date_str: str):
    rows = get_attendance(date_str)
    import csv, io
    from fastapi.responses import StreamingResponse
    output = io.StringIO()
    w = csv.DictWriter(output, fieldnames=rows[0].keys() if rows else [])
    w.writeheader(); w.writerows(rows)
    output.seek(0)
    return StreamingResponse(output,
        media_type='text/csv',
        headers={'Content-Disposition': f'attachment; filename=attendance_{date_str}.csv'})

import asyncio
 
@app.websocket('/ws/stream')
async def video_stream(websocket: WebSocket):
    await websocket.accept()
    cap = cv2.VideoCapture(0)
    try:
        while True:
            ret, frame = cap.read()
            if not ret: break
 
            faces = detector.detect(frame)
            for face in faces:
                x1,y1,x2,y2 = face.bbox.astype(int)
                crop = detector.crop_face(frame, face)
                is_live, ls = liveness.check(crop)
                if not is_live:
                    cv2.rectangle(frame,(x1,y1),(x2,y2),(0,0,255),2)
                    cv2.putText(frame,'SPOOF',(x1,y1-10),
                        cv2.FONT_HERSHEY_SIMPLEX,0.6,(0,0,255),2)
                    continue
                name, conf = recognizer.identify(crop)
                color = (0,255,0) if name!='Unknown' else (0,165,255)
                cv2.rectangle(frame,(x1,y1),(x2,y2),color,2)

                    cv2.FONT_HERSHEY_SIMPLEX,0.6,color,2)
 
            _, buf = cv2.imencode('.jpg', frame)
            await websocket.send_bytes(buf.tobytes())
            await asyncio.sleep(0.04)    # ~25 FPS
    except: pass
    finally: cap.release()

async function captureFrames() {
  const stream = await navigator.mediaDevices.getUserMedia({video:true});
  const video = document.getElementById('camera');
  video.srcObject = stream;
  const canvas = document.createElement('canvas');
  const blobs = [];
 
  for (let i=0; i<20; i++) {
    await new Promise(r => setTimeout(r, 300));  // 300ms apart
    canvas.width = video.videoWidth;
    canvas.height = video.videoHeight;
    canvas.getContext('2d').drawImage(video, 0, 0);
    const blob = await new Promise(r => canvas.toBlob(r, 'image/jpeg'));
    blobs.push(blob);
  }
 
  const form = new FormData();
  form.append('name',  document.getElementById('name').value);
  form.append('roll_number', document.getElementById('roll').value);
  form.append('class_name', document.getElementById('class').value);
  blobs.forEach((b,i) => form.append('photos', b, `photo_${i}.jpg`));
 
  const res = await fetch('http://localhost:8000/api/register',
                         {method:'POST', body:form});
  const data = await res.json();
  alert(data.success ? 'Registered!' : data.error);
}

7. Complete System Workflow — From Start to Finish
7.1  Initial Setup Workflow (One Time)
Install all packages and download models (Phase 0)
Create MySQL database and run schema SQL
Configure .env file with database credentials
Start FastAPI server: uvicorn main:app --host 0.0.0.0 --port 8000
Open registration page and register each student (15–20 photos per student)
Verify recognition works: stand in front of camera, call /api/process-frame
Deploy camera_client.py on the classroom PC connected to the camera

7.2  Daily Operation Workflow (Every School Day)
FastAPI server starts automatically (or is always running)
camera_client.py runs on classroom PC — sends a frame every 5 seconds
Students enter classroom — camera captures them automatically
SCRFD detects faces → MiniFASNet checks liveness → ArcFace identifies → MySQL records attendance
Teacher opens dashboard in browser — sees live feed with name labels and attendance list
At end of day, teacher exports CSV report from the dashboard

7.3  Adding a New Student Workflow
Admin opens registration page on laptop
Enters student name, roll number, and class
Clicks 'Start Capture' — webcam captures 20 photos automatically
System checks: if roll number already in database → rejects with error message
System checks: if face embedding matches existing student above 70% → rejects (duplicate face)
If both checks pass → saves to MySQL + updates FAISS index
Student is immediately recognizable — no server restart needed
